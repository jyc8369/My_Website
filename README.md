# jyc8369 Personal Project Showcase

[English](README.md) | [한국어](README_KO.md)

This is a simple website that shows personal projects.
It is made to look like a small project gallery, not a long resume page.

## What you can see

- Home page
- Project list
- Blog page
- Theme switch

## How to use

1. Open `main.html` in a browser.
2. Browse the project pages.
3. Click links to open the project or GitHub pages.

## Notes

- This is a static website.
- It can be hosted on GitHub Pages or similar services.
- More details are in the developer guide.
