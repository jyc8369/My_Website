# jyc8369 개인 프로젝트 쇼케이스

[English](README.md) | [한국어](README_KO.md)

이 웹사이트는 개인 프로젝트를 보여 주는 간단한 사이트입니다.
긴 소개 글보다 프로젝트를 보기 좋게 보여 주는 데 집중합니다.

## 볼 수 있는 내용

- 홈 페이지
- 프로젝트 목록
- 블로그 페이지
- 테마 전환

## 사용 방법

1. 브라우저에서 `main.html`을 엽니다.
2. 프로젝트 페이지를 둘러봅니다.
3. 프로젝트나 GitHub 링크를 눌러 봅니다.
